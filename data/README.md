# Data

## Diabetes 130-US Hospitals

- **Source**: [UCI ML Repository, id 296](https://archive.ics.uci.edu/dataset/296)
- **Files**: `diabetic_data.csv`, `IDS_mapping.csv`

### Local setup

1. Download from [UCI](https://archive.ics.uci.edu/static/public/296/diabetes+130-us+hospitals+for+years+1999-2008.zip)
2. Extract and place `diabetic_data.csv` (and optionally `IDS_mapping.csv`) in this folder

### CI/CD

The GitHub Actions workflow automatically downloads the dataset before training.
